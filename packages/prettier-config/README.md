# @t3ned/prettier-config

**Prettier configuration**

</div>

# Installation

```sh
# npm
> npm i -D @t3ned/prettier-config

# yarn
> yarn add -D @t3ned/prettier-config

# pnpm
> pnpm add -D @t3ned/prettier-config
```

# Usage

Create a `.prettierrc` file:

```json
"@t3ned/prettier-config"
```
