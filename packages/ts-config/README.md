# @t3ned/ts-config

**Typescript configuration**

</div>

# Installation

```sh
# npm
> npm i -D @t3ned/ts-config

# yarn
> yarn add -D @t3ned/ts-config

# pnpm
> pnpm add -D @t3ned/ts-config
```

# Usage

Create a `tsconfig.json` file:

```json
{
	"extends": "@t3ned/ts-config"
}
```
